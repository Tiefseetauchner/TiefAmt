# TiefAmt — Agent Instructions

## Design principles

User-defined design principles are tracked in [AGENT_DESIGN.md](./AGENT_DESIGN.md).

When working on this codebase, update `AGENT_DESIGN.md` whenever the user states a new design
principle or preference. Only record principles the user has explicitly stated — do not add
inferred conventions, general best practices, or stylistic opinions that the user did not
express.
